Reverse Words // code challenge: easy

Instructions:

Write a method to reverse the words of an input sentence. Reverse and print out each line's content as a new line to either the console.log() or the #output DOM element.

You may print the output of your solution via console.log() or insert into the #output element. If you wish to use the console.log() you'll need to have Firebug or your browser's developer console enabled. 

# Sample Input

Hello World
It's a trap
            
# Sample Output
            
World Hello
trap a It's

Use the following content of the #input pre element as the input to your method. 

I'm Mac Daddy
dead or alive
iFhone 5
So just buy
and don't ask why
you know you're gonna get an iFhone 5
