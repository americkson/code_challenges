Fibonacci Series // code challenge: easy

Instructions:

The Fibonacci series is defined as:

F(0) = 0; F(1) = 1; F(n) = F(n-1) + F(n-2) when n > 1;

Given a text file containing one positive integer n per line, print out the F(n).

If you wish to use the console.log() you'll need to have Firebug or your browser's developer console enabled. 

# Sample Input

5
12
            
# Sample Output
            
5
144

Use the following content of the #input pre element as the input to your method. 

5
8

12
